# Railway Reservation System

## Overview
This Railway Reservation System is implemented using Spring Boot microservices architecture.

## Architecture
- **Microservices**: The system is built using Spring Boot microservices, allowing for a scalable and modular design.
- **Database**: MongoDB is used as the database, providing flexibility in handling unstructured data.
- **Frontend**: The user interface is developed using React, offering a responsive and dynamic user experience.

## Configuration
- **Spring Boot**: Ensure you have the correct application properties configured in `application.properties`:
  - Server port
  - MongoDB connection details

- **MongoDB**: Make sure MongoDB is running and accessible.

- **React**: Setup the React environment with the required dependencies.


## Getting Started
1. **Clone the Repository**
   ```bash
   git clone https://github.com/saurabhvardhan001/Railway_Reservation_System.git
   cd Railway_Reservation_System
   ```
2. **Build the Application**
   ```bash
   ./mvnw clean install
   ```
3. **Run the Services**: Follow the instructions to run the Spring Boot services.
4. **Start the React Frontend**: Navigate to the frontend directory and start the React app.
5. **Access the Application**: Open a browser and access the application through the specified URL.

## Conclusion
This application demonstrates the use of modern web technologies in creating a functional railway reservation system.
